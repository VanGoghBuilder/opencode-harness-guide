## When to use
Use this when you need to check whether repository docs still match real files and current repo boundaries.

## Steps
1. Check for invented commands.
2. Check for broken or missing navigation links.
3. Check whether present facts and future plans are mixed together.
4. Check whether English and Chinese entry docs drifted apart.
5. Output only the issue list unless file edits were explicitly requested.
